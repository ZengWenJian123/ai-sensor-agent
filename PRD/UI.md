# UI 方案（v0.2）

## 1. 设计目标
- 以“快速阅读语录”为核心，降低干扰。
- 视觉风格：简洁、现代、信息优先。
- 采用 Tailwind CSS 实现统一样式系统。

## 2. 视觉基调
- 主色：中性灰 + 单一强调色（如蓝色）。
- 卡片风格：圆角、轻阴影、留白充足。
- 字体层级：
  - 一级：语录正文
  - 二级：角色名
  - 三级：来源作品/标签

## 3. 页面与组件优先级
1. 首页（今日推荐、随机语录流、换一批按钮）
2. 人物页（人物筛选 + 语录列表）
3. 搜索页（关键词输入 + 结果高亮）
4. 投稿页（语录上传表单）

## 4. 关键交互
- 语录卡片支持一键复制。
- 首页支持“换一批”随机刷新。
- 人物页支持标签过滤。
- 投稿页支持字段校验与提交结果提示。

## 5. Tailwind 规范建议
- 间距：统一使用 4/6/8 体系（`p-4`, `p-6`, `gap-4` 等）。
- 圆角与阴影：`rounded-2xl` + `shadow-sm` 为默认卡片规范。
- 主题扩展：预留深色模式（`dark:` 前缀）但可在 V1 再启用。
